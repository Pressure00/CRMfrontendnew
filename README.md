# Declarant CRM — Frontend

Современная CRM-система для декларантов и сертификатчиков.

## Технологии

- **React 18** + **TypeScript**
- **Vite** — сборка
- **Tailwind CSS** — стилизация (Windows 11 дизайн)
- **Zustand** — управление состоянием
- **React Router v6** — маршрутизация
- **Axios** — HTTP-клиент
- **Recharts** — графики
- **React Hot Toast** — уведомления
- **React Icons** — иконки

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить в dev-режиме
npm run dev

# Собрать для продакшена
npm run build

# Превью собранной версии
npm run preview
